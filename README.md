# PictureDisplayViewForAndroid
用法：


工程目录下build.gradle:

  allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
    app目录下build.gradle:
    dependencies {
    implementation 'com.github.OneZeroYang:PictureDisplayViewForAndroid:1.1.0'
}
